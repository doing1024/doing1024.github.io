* 题目
[[https://s21.ax1x.com/2024/09/20/pAMPlWD.png]]
+ 提示： *等高等底，32-24=8，24-8=16，（16+32）×2=96*
* 解题步骤
+ 设32为a
+ 设24为b
+ 设半个长方形为g
+ 设a、b之间的小三角形为c
+ 设g-a=d
+ 设g-b-c=e
+ {{{bcs}}} a+d的底为AD,a+c的底亦为AD.a+d的高为AB,a+c的高=AB
+ {{{tf}}} a+d=a+c
+ {{{tf}}} d=c
+ {{{bcs}}} d+a=c+b+e
+ {{{tf}}} d+a=d+b+e
+ {{{tf}}} a=b+e
+ {{{tf}}} e=a-b
+ {{{tf}}} e=8
+ {{{bcs}}} 32/8=4,d=c
+ {{{tf}}} x=d=c,32/x=x/4,求解得x=d=c=16
+ {{{tf}}} 2(a+d)=2 * (32+16) = 96
+ {{{tf}}} 长方形面积为96
