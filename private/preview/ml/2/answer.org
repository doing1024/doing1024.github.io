* P15填空（陈亦壮）
+ 我句式
  + 可能的表达：我不想报跳远，我想报短跑。
  + 可能的结果：好吧，我给你协调一下。
+ 你句式
  + 可能的表达：你为什么要给我报跳远？我想报短跑。
  + 可能的结果：我考虑的是集体的利益，不是你想报什么就报什么。
